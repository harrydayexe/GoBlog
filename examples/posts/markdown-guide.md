# Markdown Formatting Guide

This post demonstrates the various markdown features supported by GoBlog.

## Text Formatting

You can use **bold text**, *italic text*, and even ***bold italic text***.

You can also use ~~strikethrough~~ text.

## Lists

### Unordered Lists

- Item one
- Item two
  - Nested item
  - Another nested item
- Item three

### Ordered Lists

1. First step
2. Second step
3. Third step

## Links and Images

## Code Blocks

Inline code: `fmt.Println("Hello")`

Block code with syntax highlighting:

```python
def greet(name):
    return f"Hello, {name}!"

print(greet("World"))
```

```javascript
function greet(name) {
    return `Hello, ${name}!`;
}

console.log(greet("World"));
```
