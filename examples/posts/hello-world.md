# Welcome to My Blog!

This is my first blog post created with **GoBlog**, a static site generator written in Go.

## Why GoBlog?

I chose GoBlog because:

- It's written in Go
- It supports markdown with frontmatter
- It has syntax highlighting out of the box
- It's unopinionated and configurable
